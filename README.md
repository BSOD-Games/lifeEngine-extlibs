# lifeEngine extlibs

# Описание
Данный репозиторий содержит зависимости движка для его компиляции

# Платформы
Имеются в репозитории скомпилированые библиотеки для таких компиляторов:
* MSVC 2017 (Win32)

# Версии
* Assimp - 5.0.1
* Physics Bullet - 2.89
* FreeImage - 3.18.0
* FreeType - 2.10.1
* GLEW - 2.0.1
* RapidJson - 1.1.0
* SDL2 - 2.0.10
* Tiny C Compiler - 0.9.27
