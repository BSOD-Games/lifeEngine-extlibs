# lifeEngine extlibs

# Описание
Данный репозиторий содержит зависимости движка для его компиляции

# Платформы
Имеются в репозитории скомпилированые библиотеки для таких компиляторов:
* MSVC 2019 (x86, x64)

# Версии
* SDL2 - 2.0.12
* GLEW - 2.1.0
* STB_Image - 2.26
* STB_ImageWrite - v1.15
* RapidJSON - 1.1.0
* TMXLite - 1.2.1
